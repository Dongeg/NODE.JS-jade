<h3>简单的传值</h3>

index.jade
```
doctype html
html
	head
		meta(charset='utf-8')
		title #{course}
	body
		p hello world
```
输入命令(注意obj与后面的"之间有一个空格)
```
$ jade index.jade -P -w --obj "{'course':'jade-demo'}"
```
编译结果
```
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>jade-demo</title>
  </head>
  <body>
    <p>hello world</p>
  </body>
</html>
```
