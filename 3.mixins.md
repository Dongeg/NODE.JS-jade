<h3>mixin</h3>
最简单的栗子
```
doctype html
html
  head
    title demo
  body
    //-定义mixin
    mixin lesson(name,loves)
      p #{name}
      ul.love
        each love in loves
          li=love
    //调用mixin
    +lesson('deg',['sleep','sm','papapa'])

```
结果
```
<!DOCTYPE html>
<html>
  <head>
    <title>demo</title>
  </head>
  <body>
    <!--调用mixin-->
        <p>deg</p>
        <ul class="love">
          <li>sleep</li>
          <li>sm</li>
          <li>papapa</li>
        </ul>
  </body>
</html>
```
mixin的嵌套
```
doctype html
html
  head
    title demo
  body
    //-定义mixin
    mixin study(name,loves)
      p #{name}
      ul.love
        each love in loves
          li=love
    mixin group(student)
      h4 #{student.name}
      +study(student.name,student.love)
    +group({name:'tom',love:['qqq','www','ppp']})
```
result
```
<!DOCTYPE html>
<html>
  <head>
    <title>demo</title>
  </head>
  <body>
        <h4>tom</h4>
            <p>tom</p>
            <ul class="love">
              <li>qqq</li>
              <li>www</li>
              <li>ppp</li>
            </ul>
  </body>
</html>
```
